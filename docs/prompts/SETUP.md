# When setting up the project for 1st time
- make sure to disconnect from https://github.com/kvasdopil/nextjs repo
- create a new repo in github, ASK user for the repo name if not provided
- create a project in vercel using `vercel` cli, connect to the new gh repo, set up autodeploy
- set up a subdomain in .guskov.dev, create A record in dns using `gcloud` cli
